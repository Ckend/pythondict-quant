# 量化投资实战教程 —— Python实用宝典

许多技术投资方面的教材，经常会用几幅上涨的图来表明某些指标的用处，实际上那些上涨的图很可能只是假象。作者为了证明他所强调的指标的作用，选定了符合该指标策略的股票上升趋势图，但实际上这些策略并不一定适合全部股票，许多人被傻傻地骗了进去，血本无归。

因此，判断一个策略的好坏一定要有回测证据，要有数字证明，而非花言巧语，而本教程的最终目的，就是教你如何生成这些证据。
